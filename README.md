# Eepy Deepy
A C# Godot made 2d sokoban puzzle game. First idea was made during the MUNI FI Game Jam with the theme "Tak bojuj, ne!".
## Versions
### 1.2
Last stable release. Was submitted to the game jam.  
### 2.0
2.0 is being worked on which will bring in the full vision
- Slime walls and corners
- Smarter and more intuitive enemy AI
- Overall polishness and refactored code
- Tutorial with better UI and UX
